# Checks the operability of 'gfonts'

Checks that 'gfonts' is installed and can be used. Packages 'curl' and
'gfonts' must be installed and internet connectivity must be active.

## Usage

``` r
check_gfonts(errors = FALSE)
```

## Arguments

- errors:

  if TRUE, function is triggering errors if a condition is not
  satisfied.

## Value

TRUE or FALSE
